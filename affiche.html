Difficultés Surmontées

  • Sécurisation du LLM : Développement d'un système robuste contre les attaques de Prompt Injection ("ignore previous instructions") via des expressions régulières compilées et des clauses de sécurité système.
  • Filtrage de données massives : Migration de Pandas vers Polars pour gérer efficacement les bases IMDB lourdes. Création d'une fonction d'évaluation rapide croisant scores, années et réalisateurs.
  • Orchestration Multi-Agents : Création d'un flux d'état asynchrone avec LangGraph, obligeant le premier agent à formater strictement son output en JSON exploitable par l'agent de recherche Python.

Prochaines Étapes (En cours)

  • Base Vectorielle (RAG) : Remplacement du filtrage textuel classique par une recherche sémantique via une VectorDB (ChromaDB / FAISS) pour comprendre des requêtes comme "un film qui fait réfléchir sur le temps".
  • Machine Learning : Utilisation des données récoltées via le Dashboard Feedback pour fine-tuner un modèle de recommandation personnalisé à l'utilisateur.
